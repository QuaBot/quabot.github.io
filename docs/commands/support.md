# Support Commands

> Last edited: 1/23/2022 at 1:36PM

Commands used to create tickets etc.

#### `/add [user]`
Add a user to a ticket.

#### `/close`
Close a ticket.

#### `/delete`
Delete a ticket.

#### `/endpoll [id]`
End a poll.

#### `/endsuggestion [id]`
End a suggestion.

> [!ATTENTION]
> The Poll module is disabled.
#### `/poll [content]`
Start a poll.

#### `/quabot (suggestion) (report)`
Leave bug reports and suggestions to quabot.

#### `/reopen`
Reopen a ticket.

#### `/suggest [suggestion]`
Create a suggestion.

#### `/ticket (topic)`
Create a ticket.
