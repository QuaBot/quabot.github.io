# Management Commands

> Last edited: 5/21/2022 at 12:46PM

Configure QuaBot, lock a server down, these are the management commands.

#### `/clear [amount]`
Clear an amount of messages.

#### `/config-level`
Configure the levels module.

#### `/config`
Configure QuaBot.

#### `/poll`
Manage polls.

#### `/reactionroles`
Mangage the reaction roles.
