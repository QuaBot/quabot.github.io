# Info Commands

> Last edited: 5/20/2022 at 10:16PM

Bot status, ping, info about a user, these are the info commands.

#### `/credits`
View QuaBot's credits.

#### `/help`
Bot commands.

#### `/info`
Bot info.

#### `/ping `
Bot ping.

#### `/profile (user)`
Get a user's profile.

#### `/roles`
Server's roles.

#### `/serverinfo`
Server's info.

#### `/status`
Bot status.

#### `/user bio`
Set your profile bio.

#### `/user settings`
Configure your preferences & settings.

#### `/userinfo`
Info about a user.
